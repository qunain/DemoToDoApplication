ToDoApplication Demo

This project is a sample demo application which uses SpringBoot with Restful Microservices with Cassandra as the
DataBase and the inbuilt TomCatServer with gradle .

Cassandra version: 2.2
IntelliJ
Lombok plugin.

Run schema as given under the  resource folder.
To enable lombok plugin :
For example: Mac, In IntelliJ. got to IntelliJIDEA -> preferences ->Build,Execution,Deployemnt-> Annotation
Processors-> Look fro lombok and enable .

Run the server and use the postman collection.
